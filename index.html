Agentic RAG System with Evaluation Harness

GitHub

A question-answering agent over openFDA drug labels. It rewrites the question, pulls candidates with hybrid BM25 and dense retrieval (combined with reciprocal-rank fusion), reranks them with a cross-encoder, and runs the whole thing through a LangGraph loop with tool calls. I put it behind a DeepEval test suite and a CI gate, so every change gets scored against a plain single-shot RAG baseline.

Pipeline query rewrite · hybrid retrieval (BM25 + dense, RRF) · cross-encoder rerank · LangGraph agent + MCP tools · DeepEval CI gate · FastAPI

contextual recall 0.67 to 0.83 contextual precision 0.82 to 1.00 faithfulness 1.00

Python · LangGraph · FastMCP · FAISS · BM25 · cross-encoder reranking · DeepEval · FastAPI · Docker · GitHub Actions

Causal Inference & Uplift Dashboard

Repo available on request

I wanted the real causal effect of advertising, not just a correlation, so I worked with the Criteo uplift dataset. I adjusted for confounders using propensity-score matching, modeled uplift with a T-Learner, and validated it out of sample with the Qini metric, then dug into how the effect shifts across different segments. Everything sits behind an interactive dashboard.

Pipeline 14M-row trial · propensity-score matching · T-Learner uplift · out-of-sample Qini + heterogeneity · FastAPI dashboard

avg. treatment effect +0.82 pts (95% CI +0.80 to +0.83) 11.9M matched pairs Qini 0.088 out-of-sample

Python · scikit-learn · propensity-score matching · T-Learner uplift · Qini evaluation · FastAPI · DuckDB

TaxiCast Demand Forecasting

Repo available on request

Hourly forecasts of NYC taxi demand by borough, a week out, with p10, p50, and p90 ranges rather than a single number. I built a ladder of five models, from a simple seasonal baseline up to Chronos-T5, and compared them on a rolling-origin backtest that keeps accuracy and calibration separate. An LLM writes a short dispatch briefing on top.

Pipeline hourly trips per borough · 5-model ladder (SeasonalNaive · MSTL · LightGBM · TFT · Chronos-T5) · 4-fold rolling-origin backtest · FastAPI dashboard

Chronos-T5 WAPE 13.89% vs LightGBM 15.80% MASE 0.93 168-hour horizon

Python · Amazon Chronos-T5 · LightGBM · statsforecast (MSTL) · pytorch-forecasting · FastAPI · DuckDB
